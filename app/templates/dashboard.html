🛰️ Monitoreo Satelital

SierraViva monitorea automaticamente un radio de {{ crag.monitoring_radius_km }} km alrededor de {{ crag.name }} usando datos satelitales de:

  • NASA FIRMS — Deteccion de incendios en tiempo casi-real (VIIRS/MODIS)
  • Global Forest Watch — Alertas de deforestacion integradas
  • Sentinel-2 (Copernicus) — Imagenes multiespectrales para analisis de vegetacion (NDVI, NBR)
Los datos se actualizan automaticamente cada 6 horas (incendios) y cada 24 horas (deforestacion). Las imagenes Sentinel-2 se pueden consultar bajo demanda para cualquier rango de fechas.
